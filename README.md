# Cornhole Score Tracker 🎯

A mobile-friendly Cornhole score tracker built with Python & Streamlit, using **cancellation scoring**.

## Features
- 🏆 Cancellation scoring (standard Cornhole rules)
- 📱 Large, touch-friendly inputs designed for outdoor phone use
- 🔴🔵 Two team support with custom names
- 📜 Round-by-round history log
- 🎉 Winner announcement at 21 points

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
